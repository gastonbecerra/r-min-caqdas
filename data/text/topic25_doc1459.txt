El Big Data llegó al campo


https://www.lacapital.com.ar/campo/el-big-data-llego-al-campo-n1470067.html


2017-09-16


la tecnología big data abre una puerta de oportunidades para el análisis de datos en el campo ya que permite saber, entre otras informaciones, cuántas semillas se usan, cómo es el tipo de suelo en el que se realizará la siembra, la cantidad exacta de insumos que se necesitan en cada ambiente.es información que puede usarse para realizar una evaluación que permita proyecciones más precisas optimizando el uso del tiempo y recursos. especialistas sostienen que en la era de la información, aplicar tecnologías como big data al agro, resultan imprescindibles para reducir el uso de insumos para la producción. de esta manera, se optimizan los procesos y el trabajo se vuelve más competitivo y eficiente."con el uso de diferentes aplicaciones podemos mejorar la calidad de los alimentos, obtener información de las condiciones climáticas, el estado del suelo, los cultivos sembrados; además de ahorrar recursos naturales durante la producción", dijo un informe de oracle.así la tecnología big data abre una puerta de oportunidades para el análisis de datos, para utilizar información aplicada a la siembra permiten desde monitorear anomalías en cultivos como enfermedades, plagas o necesidad de nutrientes; hasta el procesamiento de volúmenes de datos existentes de manera simple, como series históricas de clima, mapas de suelos y rendimientos de las campañas anteriores. estas herramientas se pueden aplicar también a la actividad ganadera, como los sensores y chips que permiten monitorear la trazabilidad.
